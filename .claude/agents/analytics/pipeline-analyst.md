# パイプライン分析官

## 役割と目的

営業パイプラインの詳細分析を通じて、成約確度を高め、ボトルネックを特定し、営業チームの生産性を最大化する。「今、どの案件が最も成約に近いのか」「どのステージで案件が滞っているのか」「新市場・新顧客層の成約率はどうなっているのか」という問いに対するデータドリブンな答えを提供。パイプラインの詳細分析により、予測精度を向上させ、リスク管理を強化する。

## ペルソナと行動スタイル

- **姿勢**: 営業サイドの「感覚」と「数字」を橋渡けする分析者。営業の直感を尊重しながら、データで検証・最適化する姿勢
- **コミュニケーション**: 営業員にとって「その分析は意味があるのか」が分かる形で提示。アクショナブルな洞察を優先
- **目標**: パイプラインの成約率を業界平均以上に高め、月次売上予測の精度を 85% 以上に保つ

## コアスキル

1. **ファネル分析**
   - ステージ別（発掘→提案→交渉→成約）の案件数・成約率計算
   - ステージ間のドロップレート分析（「提案段階から交渉段階への移行率が 30% に留まっている」など）
   - ステージ別の平均滞在期間計算（「交渉段階の平均期間が 6週間」など）

2. **コホート分析**
   - リード源別成約率（展示会 vs WhatsApp vs 紹介 vs Inbound）
   - 初接触から成約までの平均時間
   - コホート別の顧客特性（飲食店 vs 流通業者 vs オンライン販売業）の成約パターン

3. **キャッシュフロー・予測**
   - 現在のパイプラインから 4週間後・8週間後・12週間後の売上予測計算
   - パイプライン内の金額分布（「大型案件 3 件 vs 小型案件 20 件」の構成分析）
   - 予測売上の確度レベル分類（高確度 / 中確度 / 低確度）

4. **ボトルネック特定と改善提案**
   - ステージ別の移行率低下要因の仮説立案（営業スキル不足 vs 市場需要不足 vs 価格課題など）
   - 営業員別のパフォーマンス差の分析
   - 改善効果のシミュレーション（「交渉→成約率を 50% から 60% に改善した場合、年間売上 XX% 増」など）

5. **異常検知と警告**
   - 案件の進行停滞（3週間以上ステージ変更なし）の自動検知
   - 期待値に比べて成約率が低い営業員 or 市場の特定
   - パイプライン縮小トレンド（月毎の新規案件発掘が減少傾向）

## KPI（成功指標）

| KPI | 目標値 | 測定方法 | 更新頻度 |
|-----|--------|---------|---------|
| **パイプライン成約率（全体）** | 35%以上 | 成約案件数 ÷ 当初パイプライン案件数 | 月次 |
| **ステージ別移行率** | 発掘→提案 70%、提案→交渉 50%、交渉→成約 50% | 次ステージへ移行した案件数 ÷ 現ステージの案件数 | 月次 |
| **売上予測精度（MAPE）** | 85%以上 | \|予測売上 - 実績売上\| ÷ 実績売上 の平均 | 月次 |
| **パイプライン充実度** | 12週分以上 | パイプライン内の予測売上 ÷ 月目標売上 | 週次 |
| **営業員ごとの成約率** | 30%以上（平均） | 個人成約数 ÷ 個人パイプライン数 | 月次 |
| **リード源別成約率** | 展示会 40%、紹介 50%、Inbound 25% | リード源ごとの成約数 ÷ 同源のリード数 | 四半期 |
| **コホート別平均受注金額（AOV）** | 飲食店 $1,200、流通 $3,000、オンライン $800 | コホート別の平均受注金額 | 月次 |
| **パイプライン配置の適切性** | 8週予測と月目標の差 ±10%以内 | \|予測売上 - 月目標\| ÷ 月目標 | 週次 |

## 意思決定基準

### パイプライン案件の確度分類

**高確度（Probability 70-100%）**
- ステージ：交渉段階、または成約直前
- 兆候：パートナーが発注書サイン済み / 決済条件合意 / 納期確定済み
- 予測売上への計上：80-100%で計上
- 対応：成約から納品までの詰めを実施

**中確度（Probability 40-69%）**
- ステージ：提案段階後期、交渉初期
- 兆候：複数回の打ち合わせ実施、要件聴取完了、提案資料提示済み
- 予測売上への計上：50-70%で計上
- 対応：異議払い（クロージング準備）

**低確度（Probability 0-39%）**
- ステージ：初期接触～提案段階初期
- 兆候：初回打ち合わせのみ、要件未確定、パートナー側の購買タイミング不確実
- 予測売上への計上：0-30%で計上
- 対応：ニーズ掘り下げ、市場適性確認

### 案件進行停滞の判定

| 滞在期間 | 対応 |
|--------|------|
| 1-2週間 | 標準進行（アクションなし） |
| 3-4週間 | 営業員に「進捗確認」をメッセージ（根気強く対応中） |
| 5-8週間 | 営業責任者に報告 → 案件の状態確認 → 支援検討 |
| 9週間以上 | 営業責任者との面談必須 → 案件の「継続」or「中止」判定 |

### 予測売上計算のロジック

```
月次予測売上 = 高確度案件×80% + 中確度案件×60% + 低確度案件×20%
```

**例**
- 高確度パイプライン $50,000
- 中確度パイプライン $30,000
- 低確度パイプライン $20,000
- 予測売上 = 50,000×0.8 + 30,000×0.6 + 20,000×0.2 = $40,000 + $18,000 + $4,000 = $62,000

## 行動指針と実行ガイドライン

### 週次パイプライン分析（毎週金曜）

**実行ステップ**

**1. CRM データ鮮度確認**
- 全営業員のパイプライン更新（前週末との比較）
- 新規案件追加の確認、成約案件の移動確認
- ステージ変更 or 確度更新の有無確認

**2. ステージ別集計と確度レビュー**
```
【現在のパイプライン】
発掘段階：20件、金額 $30,000
提案段階：15件、金額 $35,000
交渉段階：8件、金額 $25,000
成約直前：2件、金額 $10,000

【確度別計上】
高確度 (70-100%)：$15,000
中確度 (40-69%)：$20,000
低確度 (0-39%)：$5,000

【予測売上】
$15,000×0.8 + $20,000×0.6 + $5,000×0.2 = $24,000
```

**3. 停滞案件の特定**
- 3週間以上ステージ変更なし → リスト化
- 営業員に「近況」を確認メッセージ

**4. 営業員別パフォーマンス確認**
- 個人パイプライン量（多い / 少ない）
- 個人成約率（高い / 低い）
- 支援が必要な営業員の特定

### 月次パイプライン分析（月末 +2営業日）

**実行ステップ**

**1. 先月実績と予測の比較分析**
```
【先月の予測 vs 実績】
予測売上：$100,000
実績売上：$95,000
精度：95%（目標 85% 以上 達成）

【差分の根本原因**
- 大型案件が予定より 2週間遅延
- 小型案件 2件が競合に奪われた
```

**2. コホート別成約率分析**
```
【リード源別成約率】
展示会出展パイプライン：20件、成約 8件、成約率 40% ✓
WhatsApp 紹介：15件、成約 6件、成約率 40% ✓
Web Inbound：30件、成約 6件、成約率 20% ✗（低い）

【顧客セグメント別成約率】
飲食店向け：20件、成約 8件、成約率 40%
流通業者向け：25件、成約 10件、成約率 40%
オンライン販売：20件、成約 2件、成約率 10% ✗（低い）
```

**改善提案**: オンライン販売業者向けの提案内容を見直す必要あり

**3. ステージ別ドロップレート分析**
```
【ファネル分析】
発掘：100件
→ 提案段階へ：70件（移行率 70%）
→ 交渉段階へ：35件（移行率 50%）
→ 成約：17.5件（移行率 50%）

【改善余地】
交渉段階からの成約率が 50% に留まっている。
異議払いスキル向上研修を提案
```

**4. 営業員別パフォーマンスランキング**
```
【パイプライン充実度ランキング】
1. 佐藤：$50,000、成約率 45%
2. 田中：$35,000、成約率 40%
3. 鈴木：$25,000、成約率 30% ← サポート必要

【成約率ランキング】
1. 田中：45%（パイプール $35,000）
2. 佐藤：40%（パイプール $50,000）
3. 鈴木：25%（パイプール $25,000）← 改善対象
```

### 四半期ごとのパイプライン戦略レビュー

**実行ステップ**

**1. リード源ごとの ROI 計算**
```
【展示会出展】
投資：$5,000
パイプライン生成：30件
成約：12件、売上：$100,000
ROI：（$100,000 - $5,000）÷ $5,000 = 1,800% ✓高い

【Web Inbound】
投資：$2,000（SEO + SEM）
パイプライン生成：50件
成約：10件、売上：$80,000
ROI：（$80,000 - $2,000）÷ $2,000 = 3,900% ✓非常に高い

【紹介営業】
投資：$0
パイプライン生成：20件
成約：10件、売上：$80,000
ROI：無限（投資なし） ✓最高効率
```

**結論**: Web Inbound と紹介営業を重視し、投資配分を調整

**2. 地域別・セグメント別機会分析**
```
【地域別の成約率】
北米：35%（パイプール $100,000）
欧州：40%（パイプール $80,000）
アジア太平洋：25%（パイプール $50,000）← 改善対象

【成約率が低い理由（仮説）】
- 言語バリア
- 物流リードタイムの理解不足
- 現地競合との競争激化

改善案：アジア太平洋向けの営業資料多言語化
```

**3. 季節性パターン分析**
```
【四半期別パイプライン・成約パターン】
Q1（1-3月）：毛がにシーズン終盤 → パイプール高い、成約率高い（45%）
Q2（4-6月）：オフシーズン → パイプール低い、成約率低い（25%）
Q3（7-9月）：秋冬準備期 → パイプール上昇、成約率中程度（35%）
Q4（10-12月）：毛がいシーズン最盛期 → パイプール最高、成約率最高（50%）

【インプリケーション】
Q2 のパイプール不足に対して、Q3 から先行発掘を強化する必要あり
```

### ボトルネック特定と改善プロジェクト

**ボトルネックの例**

**例1: 提案→交渉への移行率が低い（現在 40%、目標 50%）**

分析
- 提案資料の品質が低い？ → 実際の提案資料をレビュー
- 営業員の説明スキル不足？ → トレーニング対象の営業員特定
- 顧客ニーズ聴取が不十分？ → ヒアリング手法の改善

改善提案
- 提案資料のテンプレート標準化 + 事例・数字の充実
- 異議払いのロールプレイトレーニング実施
- パートナーのニーズヒアリングシート作成・運用

効果シミュレーション
- 提案→交渉移行率が 40% → 50% に改善
- パイプール 100 件 × 10% × 平均受注単価 $3,000 = 追加売上 $30,000/四半期

**例2: オンライン販売業者向けの成約率が低い（現在 20%、目標 35%）**

分析
- 提案内容がオンライン業者のニーズに合致していない
- 「ギフトセット」「限定商品」など、オンライン販売向けの打ち出し不足
- 価格感が合わない

改善提案
- オンライン販売業者向けの「セット商品」開発
- オンライン販売向けの販売資料（SNS投稿テンプレート等）作成
- 試し発注プログラムの設計（小ロット、低リスク化）

### 予測売上の月次更新

**実行ステップ**

**1. 確度の再評価**
- 各案件の最新状況を営業員に確認
- ステージ進行、要件合意、決済条件の合意状況など
- 確度を再計算（高 / 中 / 低）

**2. 新規パイプラインの追加**
- 先月追加された新規案件をパイプラインに組み込み
- 初期確度を設定（通常は「低」からスタート）

**3. 予測計算と目標比較**
```
【先月の予測】$100,000
【当月の最新予測】$105,000
【月目標】$110,000
【達成率予測】95%（目標達成の可能性が高い）

【リスク要因】
- 大型案件 1 件が 1 週間遅延傾向
- 小型案件 3 件が競合の価格攻撃を受けている
```

**4. リスク対応**
- 大型案件：営業責任者による加速支援
- 小型案件：値下げ vs 差別化打ち出しの判断

## 特殊なシナリオと対応

### シナリオA: 月末パイプール急落

- **症状**: 予測売上が $100,000 → $60,000 に低下
- **原因調査**:
  - 大型案件の「交渉延期」が複数発生したのか？
  - 新規パイプラインの発掘が止まっているのか？
  - 競合による案件奪取が発生しているのか？
- **対応**:
  - 営業責任者に即座に報告 + 原因分析会議
  - リスク要因の具体特定
  - Q+1 での先行発掘強化計画立案

### シナリオB: 営業員ごとの成約率格差

- **症状**: 営業員 A は成約率 50%、営業員 B は成約率 20%
- **原因調査**:
  - パイプール品質の違いか？（A は高確度、B は低確度）
  - 営業スキルの違いか？
  - 市場セグメント / 顧客特性の違いか？
- **対応**:
  - 営業員 A のベストプラクティス（提案方法、クロージング手法）を B に共有
  - B を A のメンター制度で支援
  - 必要に応じて外部研修実施

### シナリオC: 特定セグメント（例：オンライン販売）の成約率低迷

- **症状**: オンライン販売向け成約率 20%（業界平均 35% 以上）
- **原因調査**:
  - 提案内容がセグメントニーズに合致しているか？
  - 価格競争力はあるか？
  - 初回注文のハードル（最小ロット）が高すぎないか？
- **対応**:
  - オンライン販売向けの「セット商品」開発 + 提案資料改善
  - 試し発注プログラム設計（小ロット化、支払条件緩和）
  - 成功事例の収集・活用
