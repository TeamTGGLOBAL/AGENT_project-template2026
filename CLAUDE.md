# AGENT_project-template2026

## このリポジトリについて

TG GLOBALのAI×営業プロジェクトを素早く立ち上げるためのテンプレートリポジトリ。

TG GLOBALの営業プロジェクトで確立した「AI×営業プロジェクトの進め方」を汎用化し、
どんな商材・ターゲット・国でも転用できる形に整備したもの。

## テンプレートの選び方

| パターン | 場所 | 向いている状況 |
|---|---|---|
| **A：穴埋め式** | `template/pattern-A/` | ターゲット・商材・市場がすでに決まっている。過去の営業プロジェクトに近い構造のケース |
| **B：説明と空欄式** | `template/pattern-B/` | 商材・国・業種が異なる。まず思考を整理したい段階。自由度を高く保ちたい |

迷ったらまずパターンBで考え、具体化できたらパターンAに移行することを推奨する。

## 使い方

1. `template/pattern-A/` または `template/pattern-B/` を新規リポジトリにコピーする
2. リポジトリ名は `AGENT_[商材]-[市場]-[年]` の形式を推奨
3. `CLAUDE.md` のミッション・勝利の定義から埋め始める
4. フェーズ0（偵察）を完了してからフェーズ1に進む
5. 各フェーズ終了時にCLAUDE.mdを更新して進捗を反映する

## フレームワークの核心

- **価格で戦わず、比較される土俵を変える**
- **意思決定者に直接届けるルートを設計する**
- **Go/No Goで判断しながら進む。止まる条件を先に決める**
- AIの役割は4つ：情報分析官・市場リサーチャー・提案設計者・コミュニケーター

## エージェントへの共通指示

各プロジェクトでAIエージェントに期待する役割：

1. **情報分析官** — 入手した情報（LINE履歴・メール・展示会メモ等）から判断材料を抽出する
2. **市場リサーチャー** — ターゲット市場のトレンド・競合動向・規制を調査する
3. **提案設計者** — 誰に・いつ・何を・どう提案するかを設計する
4. **コミュニケーター** — 相手の言語・文化に合わせた提案文・メッセージ案を作成する

## TGプリンシパル

「TGプリンシパル」とは、TG GLOBALの社内戦略ドキュメント群です。

以下が含まれます。

- Mission / Vision / Value
- 経営戦略
- 海外卸方針
- ブランド思想
- 社内標準

ユーザーが以下のように言った場合は、TGプリンシパルを優先的に参照してください。

- 「TGプリンシパルを参照して」
- 「TGプリンシパルではどう定義してる？」
- 「TGプリンシパル基準で考えて」

TGプリンシパルを使用する際は、ターミナルでclaudeを起動したあとに、`@[TGプリンシパルのローカルパス]` を実行してください。パスは社内で管理しています。

## TG GLOBAL 独自ツール

### CS_EmailManager

TG GLOBALが自社開発したGoogleスプレッドシートのアドオン。カスタマーマーケティング施策で活用する。

**主な機能：**
- **一括送信** — リストに対してメールを一括配信
- **テンプレート管理** — メール下書きをテンプレート化して再利用
- **送信間隔の調整** — 送信アクセス間隔を設定してスパム判定を回避
- **開封トラッキング** — 受信者の開封有無を確認
- **ID管理** — 送信先の顧客IDを管理

接触シナリオや提案設計の段階でメール配信が有効な手段かどうかを検討する際は、このツールの活用を提案すること。市販の有料メール配信サービスに匹敵する機能を持つ。
