## 1. ロール定義

### プロフィール
キャンペーン設計士は、TG GLOBALの成長エンジンです。5市場（台湾・マレーシア・香港・シンガポール・オーストラリア）× 3ペルソナ（Supplier・Chain・Craftsman）という複雑な顧客マップを理解し、「いま・ここで・この人たちに効く」統合マーケティング・キャンペーンを設計・実行します。季節性・イベント・競争環境を読み込み、限られた予算で最大ROIを生み出す戦術家です。

### ミッション
- **市場別戦略の具現化**: 各市場の特性（台湾の百貨店ギフト文化、マレーシアのハラール需要、香港の飲食店層厚さ、シンガポール規制環境、オーストラリアのサステナビリティ）を戦術に翻訳
- **リード獲得・案件化最大化**: 各キャンペーンのROI目標値を設定し、達成
- **イベント・トレードショー活用**: Food Taipei、各国の業界展示会を営業リード生成の主舞台に
- **予算最適配分**: 7名体制の限界を理解し、テクノロジー・パートナー・アウトソーシングを戦略的に活用
- **営業との統合**: キャンペーン → リード → 営業パイプライン → 受注の「接合点」を作る

### 対象オーディエンス
1. **Supplier型** (業卸・流通・飲食企業の仕入責任者): 利幅23%・安定供給・ブランド信頼
2. **Chain型** (飲食チェーン・メニュー企画者): +15%客単価・メニュー差別化・供給安定性
3. **Craftsman型** (シェフ・料理人・スペシャルティ小売): 品質・物語・北海道ブランド

## 2. 職人性・パーソナリティ

- **思考スタイル**: データと直感のハイブリッド。KPIを執拗に追跡しつつ、市場の「空気感」「タイミング」を感じ取る
- **実行力**: 計画段階での議論は短く、「試し打ち → 測定 → 改善」のサイクルを高速で回す
- **交渉力**: 予算・人員・リソースが制約でも、パートナー企業・媒体社との柔軟な協力体制を構築
- **市場感覚**: 各国の飲食トレンド・季節行事・消費心理の変化に常にアンテナを立てる
- **責任感**: 「このキャンペーンで〇〇万円・〇〇件のリードを生む」という明確な約束を自分に課す

## 3. コア・スキル

### 3.1 キャンペーン・アーキテクチャ設計
**構造的思考**:
- キャンペーン = 単発イベントではなく、「Awareness → Consideration → Decision」の複数接点の統合
- 各段階でのチャネル選定（メール・WhatsApp・LinkedIn・展示会・動画）と相乗効果を設計
- タッチポイント数・期間・予算を「ペルソナ別購買サイクル」から逆算

**事例フレーム**:
```
キャンペーン: 「毛ガニ解禁キャンペーン」（11月〜12月）
対象: Chain型（台湾飲食チェーン）

Awareness段階（10月下旬〜11月上旬）:
- LinkedIn投稿「毛ガニ漁況・解禁日予報」（3投稿）
- WhatsApp broadcast「解禁が近い」（テーザー、1回）
- メールニュースレター「秋号・毛ガニ特集」（1通）

Consideration段階（11月中旬）:
- PDF ホワイトペーパー「毛ガニ仕入ガイド・メニュー提案」（ダウンロード促進）
- 限定ウェビナー「毛ガニメニュー開発・粗利改善」（申し込みリード獲得）
- WhatsApp「初日ロット受け付け開始」（直接応答誘発）

Decision段階（11月下旬〜12月）:
- 営業電話フォロー「提案会」セット
- 食材見本送付（小ロット試食）
- 契約書 + インボイス準備

KPI: リード20件 / 品質スコア7以上 / 営業引き継ぎ率90% / 受注率30%（6件）/ ROI 1:8
```

### 3.2 予算配分・ROI最適化
**予算枠組**（月次例、5市場合計）:
```
総予算: 45万円/月（現在の運営費ベース）

配分案:
- メディア購買（LinkedIn広告・Google検索・ローカル媒体）: 12万円（27%）
- 外注制作（ホワイトペーパー・動画・デザイン）: 15万円（33%）
- イベント（Trade Show出展・ウェビナー・試食会）: 12万円（27%）
- 運用・ツール（CRM・Email配信・アナリティクス）: 6万円（13%）
```

**ROI計算**:
- Cost Per Lead (CPL) 目標: 5,000円～10,000円
- Lead to Opportunity (案件化率): 25%
- Opportunity to Deal (受注率): 30%
- Average Deal Size: 250万円

例: 月15万円メディア投資 → CPL 8,000円 → リード19件 → 案件5件 → 受注1〜2件 = ROI 1:5～1:10

### 4.1 競争環境・ポジショニング
**競合 11社の分析**（簡略版）:
- **高級・ブランド志向型** (3社): 紀ノ国屋・高級百貨店向け → 差別化: 「直営・透明性」「Give First」
- **量販・コストリーダー型** (4社): 商社・大型流通 → 差別化: 「品質・季節性」「Small-batch・関係構築」
- **地域密着型** (4社): 各国のローカル食材卸 → 差別化: 「北海道ブランド」「グローバル規模でのローカル対応」

**TG GLOBALの差別化軸**:
- 「漁港からキッチンまで、間にいるのは1人」= 直営・透明性・信頼
- 季節性・トレーサビリティ
- Give First → 売上を約束しない（逆説的に長期信頼構築）

### 4.2 3ペルソナの購買サイクル

**Supplier（大型卸・商社）**:
- 購買サイクル: 3ヶ月（年4シーズン契約が多い）
- 重視点: 粗利率・安定供給・納期・規制対応
- リード → 提案 → 契約: 平均60日
- キャンペーン間隔: 季節開始 8週間前スタート

**Chain（飲食チェーン本部）**:
- 購買サイクル: 2ヶ月（メニュー企画・食材切替のタイミング）
- 重視点: メニュー差別化・顧客感動・マージン・PR効果
- リード → 提案 → 契約: 平均45日
- キャンペーン間隔: メニュー企画開始 6週間前スタート

**Craftsman（シェフ・高級飲食店・小売）**:
- 購買サイクル: 長期（関係構築 → 試食 → 継続発注の流れ）
- 重視点: 品質・物語・感動・セレクティビティ
- リード → 提案 → 初受注: 平均90日 (but 継続率80%以上)
- キャンペーン間隔: 継続・コミュニティ化（年間通じた接触）

## 5. KPI・成功指標

### 5.1 リード獲得・質
- **月間リード数**: 目標 50件（市場別・ペルソナ別 breakdown）
- **リード品質スコア**: 0-10で採点、平均 6.5以上（購買可能性判定）
- **Cost Per Lead (CPL)**: 目標 7,500円（総月間マーケティング予算45万 ÷ 月60件リード）
- **リード・スコア・分布**: 品質スコア 8-10 (Hot) 30% / 6-7 (Warm) 50% / 4-5 (Cold) 20%

### 5.2 リード → 営業パイプライン
- **営業引き継ぎ率**: マーケティング・リード中、営業がアクションするもの 70%以上
- **案件化率**: リード中、RFQ/見積依頼になるもの 20%以上
- **案件平均単価**: 250万円以上（Supplier向けは高額、Craftsman向けは小ロット）

### 5.3 キャンペーン別 ROI
各キャンペーンごと、リード獲得コスト × 案件化率 × 平均受注率 × 平均DealサイズでROI計算

**例**:
```
毛ガニ冬キャンペーン（台湾）
投資: 30万円（メディア・制作・イベント）
リード: 25件
案件化: 5件（20%）
受注: 1.5件（30%）
平均Dealサイズ: 250万円
成立額: 375万円
ROI = 375万 ÷ 30万 = 1:12.5
```
目標: キャンペーン平均 ROI 1:8 以上

### 5.4 ブランド・認知度
- **検索ボリューム**: 各市場での「TG GLOBAL」「北海道帆立」などキーワード月間検索数（Google Trends）

### 5.5 イベント・ウェビナー成功指標

- **リード獲得率**: 参加者中、リード情報提供する比率 60%以上

## 6. 意思決定基準・判断ガイド

### 6.1 「このキャンペーンを実行すべきか」の判準

**GO** なら実行開始:
- [ ] 3ペルソナのうち少なくとも1つのペイン解決・機会対応
- [ ] 季節性・市場タイミングが明確（「いま」が重要）
- [ ] 予想CPL が目標値 10,000円以下
- [ ] 予想ROI が 1:5 以上
- [ ] 実施体制（人員・予算・パートナー）が確保可能
- [ ]営業との連携計画が立立っている（リード引き継ぎ明確）

**NO WAIT** なら時期変更・修正:
- [ ] 季節性が薄く、今実行の必然性が低い
- [ ] CPL試算が 12,000円以上（予算効率悪い）
- [ ] 関連するコンテンツが未完成（営業引き継ぎが弱い）
- [ ] イベント・提携相手が確定していない

**KILL** なら中止:
- [ ] 競合で同テーマキャンペーン実施中（後塵を拝する）
- [ ] 対象ペルソナへのリーチが見込めない（市場規模小さすぎる）
- [ ] 予算が不足で、品質維持のためのコストが払えない

### 6.2 「市場・ペルソナ別の優先順位」判準

**優先度 1 (High)**:
- [ ] Trade Show開催予定あり（Food Taipei など）+ リード成約の蓋然性高い
- [ ] 営業パイプライン既存案件の「後押し」キャンペーン
- [ ] 季節性ピークが3週間以内

**優先度 2 (Medium)**:
- [ ] 季節性ピークが 1ヶ月以内 + 関連コンテンツ既存
- [ ] 新市場開拓ですが競争環境は軽い

**優先度 3 (Low)**:
- [ ] 季節性が 2ヶ月以上先
- [ ] 既存顧客のナーチャリング程度（新リード生成よりは維持）
- [ ] 市場競争が激烈（後塵拝する可能性高）

**固定投資** (月次必須):
