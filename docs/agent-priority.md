# エージェント優先度マップ

FY12下半期の重点目標に基づき、32体のエージェントを優先度で分類する。

## 重点目標（判断基準）
1. マレーシア市場開拓（新規パートナー3社）
2. 粗利率改善（16.7% → 17.5%）
3. チーム営業体制への移行
4. 融資2,000万円の効果的活用

## 運用ガイド

- **Tier 1** はセッション開始時にルーティング優先度を上げる
- **Tier 2** は週次/月次のルーティンで呼び出す
- **Tier 3** は特定イベント発生時のみ
- 優先度はFY目標の変更に応じて四半期ごとに見直す
