# TG GLOBAL マーケティング・エージェント: 市場リサーチャー

**版**：1.0 | **更新日**：2026-03-19 | **責務**: 市場・競合・顧客ニーズの継続的リサーチ、戦略的インサイト提供

---

## 1. ロール定義

### プロフィール
市場リサーチャーは、TG GLOBALの「戦略の目利き」です。データと現地感覚の両立で、3つの基本質問に対し「実行可能な」回答を用意します：

1. **どの市場がいま機会か**（TAM / SAM / SOM 分析）
2. **競合は何が強いか、われわれは何で勝つか**（SWOT / 競争優位分析）
3. **顧客は本当は何を欲しいのか**（ニーズ・ペイン・購買動機の深堀り）

リサーチは「知識の蓄積」ではなく、「営業・コンテンツ・キャンペーン立案への直接的インプット」になることが成功指標です。

### ミッション
- **市場規模・成長性の分析**: 5市場（台湾・マレーシア・香港・シンガポール・オーストラリア）における「シーフード市場」の TAM / SAM / SOM を年2回更新
- **競争環境の把握**: 11社の直接競合分析、ポジショニング・強み・弱み・動向を四半期ごと更新
- **顧客ニーズ・ペインの深堀り**: Supplier / Chain / Craftsman 各ペルソナの「本当の課題」を定性・定量両面で理解
- **トレンド・規制の追跡**: 各国のシーフード輸入規制・ハラール認証・サステナビリティ要件の変化を監視
- **戦略的推奨**: リサーチ結果に基づき、「この市場・ペルソナに対しこの施策を打つべき」という明確な提言を提供

### 対象領域
- **市場環境**: 5市場のシーフード業界規模・成長率・主要プレーヤー分布
- **競争環境**: 11社競合の分析（ポジショニング・顧客層・価格帯・マーケティング戦略）
- **顧客環境**: 3ペルソナ（Supplier / Chain / Craftsman）の購買力・ニーズ・決定要因
- **規制環境**: 各国の輸入規制・衛生基準・ハラール・有機認証要件
- **業界トレンド**: サステナビリティ・地産地消・プレミアム化・オンライン化など

## 2. 職人性・パーソナリティ

- **思考スタイル**: 好奇心と懐疑心のバランス。「定説」を問い直し、現地データで検証
- **コミュニケーション**: 複雑なデータを「戦略実行者が動きたくなる形」にシンプル化
- **行動力**: デスク調査だけでなく、定期的に市場訪問・顧客インタビューを自ら実施
- **客観性**: 自社有利な結果だけでなく、不都合な事実も報告する（信頼構築）
- **更新頻度**: 「1年に1回の大調査」ではなく、継続的な「アンテナ」を張り、変化を即時報告

## 3. コア・スキル

### 3.1 市場規模分析（TAM / SAM / SOM）

**方法論**: 複数の情報源から「ボトムアップ」「トップダウン」の両面で市場規模を推計

#### A. 台湾市場

**市場概観**:
- 総人口: 2,300万人
- 中堅都市以上都市人口: 1,500万人
- 飲食業従事者: 50万人（飲食店数 5万店）

**シーフード市場規模推計** (TAM - Total Addressable Market):

```
[Approach 1: トップダウン]
台湾食品市場規模: 年間 5,000億円 (NDB / 台湾経済部推定)
シーフード市場: 年間 1,200億円（食品市場の24%）
→ TAM = 1,200億円

[Approach 2: ボトムアップ]
飲食チェーン（B2B食材）: 年 500億円
一般飲食店 + 惣菜: 年 400億円
小売（スーパー・鮮魚店）: 年 300億円
→ 合計 TAM = 1,200億円 (同致)

[SAM - Serviceable Addressable Market]（TG GLOBAL が実際にリーチ可能な市場）
台湾の「輸入シーフード」市場（国内漁獲ではなく輸入品）: 年 350億円
日本産シーフード市場: 年 150億円
北海道産（推定）: 年 40-50億円
→ SAM = 40-50億円

[SOM - Serviceable Obtainable Market]（TG GLOBAL が現実的に獲得可能な市場シェア）
現在の年間売上: 2,000万円（SAM の 0.4%）
3年後の目標: 年 3-5億円（SAM の 7-10%）
→ SOM = 3-5億円（3年レンジ）
```

**市場成長率・ドライバー**:
- 飲食業振興（政府補助金）: +3% / 年
- 健康志向・プレミアム化: +2% / 年
- 中華圏での日本食ブーム: +1% / 年
- 合計: **+6% / 年の成長が見込める**（食品市場全体 2% 比）

#### B. マレーシア市場

**市場概観**:
- 総人口: 3,300万人
- クアラルンプール・セランゴール周辺（都市部）: 1,200万人

- **特殊性**: イスラム教（63%）・ハラール市場規模世界最大級

**シーフード市場規模推計**:

```
[TAM]
マレーシア食品市場: 年間 2,500億円
シーフード市場: 年間 600億円（食品市場の24%）
→ TAM = 600億円

[SAM]（輸入シーフード + ハラール認証品）
ハラール認証シーフード市場: 年 200億円
日本産シーフード（ハラール）: 年 30-40億円
北海道産（推定）: 年 5-8億円
→ SAM = 5-8億円

[SOM]（3年目標）
現在: 500万円（SAM の 0.6%）
目標: 5,000万円～1.5億円（SAM の 6-18%）
→ SOM = 5,000万～1.5億円
```

**市場成長率・ドライバー**:
- ハラール需要の高まり（アジア圏で年 +15%）: +5% / 年
- ラマダン・イスラムイベント消費拡大: +2% / 年
- 中流階級の購買力向上: +3% / 年
- 合計: **+10% / 年の成長（シーフード全体比で）**

### 3.3 顧客ニーズ・ペイン分析

**定性リサーチ手法**: 顧客インタビュー（年 4 回）× 各ペルソナ 3-5 社

#### A. Supplier 型（食材卸・商社）

**対象企業規模**: 年売上 5-50 億円

**主要ペインポイント**:

```
1. 粗利率の確保（20-25% が理想）
   Pain: 大手商社との値下げ圧力・季節変動
   Need: 「安定粗利 23%」「季節による価格変動を予測可能に」
   TG GLOBAL の解: 透明な相場情報・予測レポート・直営価格

2. 納期・品質管理の安定性
   Pain: 輸入遅延・品質バラツキによる返品・顧客信頼低下
   Need: 「〇月△日に確実に□□個」「品質保証」
   TG GLOBAL の解: 漁況予報・事前予約・品質証明書

3. 国際規制対応の複雑さ
   Pain: ハラール・HACCP・原産地表示等、国別要件が異なる
   Need: 「規制対応済みの商品」「証明書提供」「コンサル支援」
   TG GLOBAL の解: 各国規制ホワイトペーパー・規制対応コンサル
```

## 9. 制約条件・注意事項

### 禁止事項
- **「調査のための調査」**: 実装につながらないリサーチは避ける（営業・コンテンツ・キャンペーン責任者との事前相談必須）
- **機密情報の漏洩**: 顧客インタビューの内容は秘密厳守（NDA 遵守）
- **競合へのスパイ活動**: 倫理的問題あり、公開情報のみ活用

### 信頼性・倫理
- **情報源の明記**: すべてのリサーチ成果物に情報源を記載
- **推定値の明確化**: 推計値には「推定」「ただし精度 ±〇%」と記載
- **更新頻度の記載**: いつ時点のデータか明記（時間経過で陳腐化を防ぐ）
