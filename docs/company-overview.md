## 競争優位性

### 差別化要因
1. **産地直送モデル**: 豊洲市場を経由せず、北海道の漁港から直接輸出
2. **小口対応**: 300kgから取引可能
3. **CEO直接対応**: 社長自らWhatsAppで顧客対応
4. **トレーサビリティ**: 漁港名、出荷日、船名まで追跡可能
5. **価格競争力**: 中間マージンを削減し、高品質を低価格で提供

### サプライチェーンの優位性
```
【従来の日本産シーフード】
北海道の漁港 → 苫小牧港 → 豊洲市場（手数料5.5%） 
→ 仲卸（マージン15-25%） → 輸出業者（マージン5-10%） 
→ 現地卸（マージン18-25%） → キッチン
= 間に4社、CIF $16-20/kg

【TG GLOBAL】
北海道の漁港 → 苫小牧港 → TG GLOBAL（1人） → キッチン
= 間に1人、CIF $11-12/kg
```

## リスク管理

### 主要リスク
1. **在庫リスク**: 鮮度が命。短期で売れなければ価値急落
2. **信用リスク**: 海外取引での回収遅延・不履行
3. **為替リスク**: 輸出比率が高いため為替変動が利益に直結
4. **物流リスク**: 航空便・船便の遅延は品質劣化に直結

### 対策
- 需要予測の精度向上
- 信用条件の明文化、前金化の推進
- 価格改定タイミングとヘッジ戦略
- 複数ルート確保と輸送パートナーの分散
